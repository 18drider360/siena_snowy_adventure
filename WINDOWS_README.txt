              SIENA'S SNOWY ADVENTURE - Windows Edition
================================================================================

Thank you for downloading Siena's Snowy Adventure!

================================================================================
INSTALLATION
================================================================================

1. Extract the ZIP file to a location of your choice
   (e.g., C:\Games\SienaSnowyAdventure)

2. Open the extracted folder "SienaSnowyAdventure"

3. Double-click "SienaSnowyAdventure.exe" to launch the game

That's it! No installation required.

================================================================================
SYSTEM REQUIREMENTS
================================================================================

- Operating System: Windows 10 or Windows 11 (64-bit)
- Memory: 2 GB RAM minimum
- Graphics: OpenGL 2.0 compatible graphics card
- Storage: 100 MB free space
- Display: 1280x720 resolution or higher recommended

================================================================================
CONTROLS
================================================================================

Arrow Keys or WASD - Move
Space Bar - Jump
Left Shift - Roll/Dash
ESC - Pause/Menu

================================================================================
GAME SAVES
================================================================================

Your game progress is automatically saved in:
C:\Users\<YourUsername>\.siena_snowy_adventure\

This folder contains:
- save_data.json - Your game progress
- scoreboard.json - Your local scoreboard

================================================================================
ONLINE FEATURES
================================================================================

The game includes online features:
- Global leaderboard for level completion times
- Automatic score submission when you complete levels

An internet connection is recommended but not required to play.

================================================================================
UPDATES
================================================================================

This Windows version does not include automatic updates.

To update to a new version:
1. Visit: https://github.com/18drider360/siena_snowy_adventure/releases
2. Download the latest Windows release
3. Extract and replace your current game folder

Your save files will be preserved during updates.

================================================================================
TROUBLESHOOTING
================================================================================

Problem: "Windows protected your PC" warning when launching
Solution: Click "More info", then "Run anyway"
         This warning appears because the game is not code-signed.
         The game is safe to run.

Problem: Antivirus flags the game as suspicious
Solution: Add an exception for SienaSnowyAdventure.exe in your antivirus
         This is a false positive common with PyInstaller applications.

Problem: Game won't launch or shows error about missing DLLs
Solution: Install Visual C++ Redistributable from Microsoft:
         https://aka.ms/vs/17/release/vc_redist.x64.exe

Problem: Game runs slowly or has graphics issues
Solution: Update your graphics drivers
         Ensure your system meets minimum requirements

Problem: No sound
Solution: Check Windows volume mixer
         Ensure your audio device is working
         Try adjusting audio settings in the game's OPTIONS menu

================================================================================
UNINSTALLING
================================================================================

To uninstall:
1. Delete the game folder (e.g., C:\Games\SienaSnowyAdventure)
2. (Optional) Delete save folder: C:\Users\<YourUsername>\.siena_snowy_adventure
