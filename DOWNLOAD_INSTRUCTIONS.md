# How to Download and Play Siena's Snowy Adventure

## Step 1: Download the Game

1. Click the download link I sent you
2. Click the **Download** button (Google Drive shows it in top-right)
3. Wait for the download to finish (225MB - about 2-5 minutes)
4. The file `SienaSnowyAdventure-Mac.zip` will be in your Downloads folder

## Step 2: Extract the Game

1. Go to your **Downloads** folder
2. Find `SienaSnowyAdventure-Mac.zip`
3. **Double-click** the zip file to extract it
4. A file called `SienaSnowyAdventure.app` will appear

## Step 3: First Launch (Important!)

**You'll see a security warning on first launch. This is normal!**

### If you see: "Apple could not verify..."

**Here's how to open it:**

1. **Right-click** (or Control-click) on `SienaSnowyAdventure.app`
2. Select **"Open"** from the menu
3. A dialog appears - click **"Open"** again
4. The game launches! üéÆ

### Alternative Method:

1. Try to open the app normally (you'll get the warning)
2. Go to **System Settings** ‚Üí **Privacy & Security**
3. Scroll down - you'll see a message about the blocked app
4. Click **"Open Anyway"**
5. Click **"Open"** to confirm

## Step 4: Play!

After the first launch, you can just **double-click** the app to play. No more warnings!

---

## Why the Security Warning?

The app isn't "code-signed" (which costs $99/year for developers). This is normal for indie games and small projects. The game is completely safe - it just plays music, shows graphics, and saves your scores!

## Troubleshooting

### Game won't open at all
- Make sure you extracted the zip file (don't try to run it from inside the zip)
- Try redownloading if the file is corrupted

### No sound
- Check your Mac's volume
- Check System Settings ‚Üí Sound ‚Üí make sure app isn't muted

### Can't connect to online leaderboard
- Check your internet connection
- The leaderboard may take a few seconds to load

### Game is laggy
- Close other applications
- Restart the game
